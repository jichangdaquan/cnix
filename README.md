CNIX平台由ssrcloud升级而来，被用户亲切称为"薯条"。支持SS、SSR、V2ray等主流代理协议，适配Surge客户端。采用IPLC/IEPL专线网络，保障网络拥堵时段仍可稳定观看8K超高清YouTube视频，并设有优化游戏体验的专用加速通道。
